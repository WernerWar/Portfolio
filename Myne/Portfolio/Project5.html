Game Jam in a group with C# in Unity

This is a Game Production Project where I was tasked to make a Game with Game Jam rules in a group using C# in Unity with the theme "The more you have, the worse it gets"

We decided to make a 2D Tower Defense game with blocks you can build towers on that shoots enemies that spawn every wave, and every wave more enemies spawn and become harder to kill
